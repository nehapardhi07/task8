📊 Sales Performance Dashboard

📝 Project Overview

This project focuses on building a basic interactive dashboard to analyze sales performance by Product, Region, and Month.
The dashboard helps in identifying trends, comparing regional performance, and understanding category-wise sales distribution.

🎯 Objective
    
		 To design a clean and interactive dashboard that visually represents:
     
		 Monthly Sales Trends
		 
		 Region-wise Sales Performance
		 
		 Category-wise Sales Distribution
		 
		 Profit Analysis

🛠️ Tools & Technologies

Power BI  – Dashboard creation

Python (Pandas) – Data cleaning 

CSV Dataset – Superstore Sales data


📂 Dataset Information

File Name: Superstore_Saless.csv

Columns Used:

Order Date

Region

Category

Sales

Profit

📊 Dashboard Components


1️⃣ Line Chart – Sales over Months (Month-Year format)

2️⃣ Bar Chart – Sales by Region

3️⃣ Donut Chart – Sales by Category

4️⃣ Slicer/Filter – Region or Category

5️⃣ Highlighted top-performing areas using colors

🔎 Key Insights

The highest sales were recorded in peak seasonal months.

The West region generated the highest overall revenue.

Technology category contributed the maximum sales.

Profit trends varied across regions and categories.
